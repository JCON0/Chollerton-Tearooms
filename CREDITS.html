SECURITY CONSIDERATIONS

During the creation of any website, it is critical to consider any potential security risks which may be presented on both the user and company sides. As stated by Bassi, 2023, a few things to be wary of are: the maintenance of software, SQL injection, Cross Site Scripting, error messages, data validation and vulnerabilities. All of the stated factors to consider are key areas in which hackers can exploit code and gain access to potentially sensetive information.

According to the Canadian Centre for Cyber Security, 2022, a few methods to avoid exploitation and hacking is to: consider a secure software architecture, this can be done by seggregating web service components, protecting internet facing web services and isolating administrative interfaces. Furthermore, it is key to consider appropriate access limitation, authentication methods, ensure the use of secure sessions, input validation and secure configuration and operations.
